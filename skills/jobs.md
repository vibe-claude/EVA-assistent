# Логика джобов

Для curl-шаблонов читай `notion.md`.

**Принцип:** Notion — источник правды. При каждом джобе сначала синхронизация, потом работа с данными.

---

## work-morning (рекомендуемое расписание: 08:45 пн-пт)

**ВАЖНО: вывести брифинг напрямую в ответ.**

1. `source .env && TODAY=$(date +%Y-%m-%d) && YESTERDAY=$(date -d yesterday +%Y-%m-%d)`
2. Выполнить **полную синхронизацию** из `notion.md` (читать Notion → обновить tasks/active.md)
3. Из обновлённого `tasks/active.md` выбрать задачи по фильтрам:
   - **Просроченные:** статус "В работе", дата < сегодня
   - **На сегодня:** статус "В работе", дата = сегодня
   - **В работе без даты:** статус "В работе", дата не указана
   - **На контроле:** статус "Контроль!!!" (все)
4. Запросить платежи на сегодня из Notion (если настроена DB платежей)
5. Если настроен IKIGAI — прочитать `goals/ikigai.md`

Вывести брифинг:
```
☀️ Доброе утро!

⚠️ Просроченные:
1. Задача

🗒 На сегодня:
2. Задача

🔄 В работе:
3. Задача

👁 На контроле:
4. Задача
```

**Нумерация сквозная.** Блоки без данных — пропустить.
Если данных нет совсем — молчать полностью.

## work-check (рекомендуемое расписание: 11:30 и 14:30 пн-пт)

**ВАЖНО: только список задач, никаких пояснений.**

1. Выполнить **полную синхронизацию** из `notion.md`
2. Из `tasks/active.md` выбрать:
   - На сегодня и в работе без даты
3. Если нет задач — молчать
4. Если есть:
```
🗒 На сегодня:
1. Задача

🔄 В работе:
2. Задача
```

---

## work-evening (рекомендуемое расписание: 18:00 пн-пт)

**ВАЖНО: вывести итог напрямую в ответ.**

1. `source .env && TODAY=$(date +%Y-%m-%d)`
2. Синхронизация из `notion.md`
3. Получить задачи, выполненные сегодня
4. Прочитать `wiki/log.md` — записи за сегодня
5. Переместить выполненные из `tasks/active.md` в `tasks/archive/YYYY-MM.md`
6. Вывести итог:
```
📊 Итог дня — <дата>

✅ Выполнено: X задач
1. Задача

📌 Осталось открытых: Y

🗂 Документы за сегодня:
📄 <название> — <суть>

💬 Оценка дня: <2-3 предложения>
```
7. Очистить `today-tasks.md`

## evening-reflection (рекомендуемое расписание: 20:30 ежедневно)

1. Написать:
```
Как прошёл день?
```
2. Ждать ответа.
3. Записать в дневник:
```bash
echo "" >> memory/diary.md
echo "## [$(date +%Y-%m-%d)]" >> memory/diary.md
echo "<ответ пользователя>" >> memory/diary.md
```
4. Если воскресенье — сделать анализ недели (прочитать diary.md за 7 дней) и задать вопрос:
```
💬 Что одно изменение сделает следующую неделю лучше?
```

## dream-system (ночная консолидация, рекомендуемое расписание: 03:00)

Ночная консолидация памяти:
1. Прочитать `memory/lessons.md`
2. Прочитать последние записи из `wiki/log.md` (за последние 3 дня)
3. Найти паттерны, повторяющиеся темы
4. Записать инсайт в `memory/lessons.md` если есть что-то важное

Молчать полностью — не отправлять сообщений пользователю.
