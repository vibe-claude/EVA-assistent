## Создать задачу
Триггеры: "добавь задачу", "создай задачу", "запомни", "поставь задачу"
- Название — обязательно
- Дата — если не сказано, ставить сегодня
- Подтвердить: "✅ Задача создана: <название>"
→ curl-шаблон в `notion.md`

## Закрыть задачу
Триггеры: "закрой задачу", "выполнено", "готово по <название>", "N выполнено"
Обновляет статус на "Выполнено".
→ curl-шаблон в `notion.md`

## Показать задачи
Триггеры: "что на сегодня", "покажи задачи", "что открыто"
Читает из Notion, выдаёт актуальный список.
→ curl-шаблон в `notion.md`

## Напомнить в конкретное время
Триггеры: "напомни мне в 15:00", "напомни в 17 часов <о чём>"
Создаёт задачу в Notion с полем Время.
→ curl-шаблон в `notion.md`

## Обработать документ
Триггер: входящий файл или фото в Telegram
→ читай `docs.md`

## Вопросы о датах и документах

Триггеры:
- "когда получили <документ>", "когда был <акт>", "есть ли <документ> по <объекту>"

Алгоритм:
1. Проверить wiki:
```bash
grep -ri "<объект или тип документа>" wiki/log.md
grep -rl "<объект>" wiki/ --include="*.md"
```
Прочитать найденные страницы.
2. Дать ответ. **Никогда не говорить "нет данных" не проверив wiki.**

## Найти информацию по объекту

Триггеры: "найди всё по <объект>", "что есть по <объект>", "справка по <объект>"

Алгоритм:

**Шаг 1 — Notion (задачи):**
```bash
source .env
# Запросить задачи с фильтром по названию объекта
```

**Шаг 2 — Wiki (история, документы):**
```bash
grep -rl "<slug объекта>" wiki/ --include="*.md"
```
Прочитать найденны�� страницы.

Блоки без данных — пропустить. **Никогда не говорить "нет данных" не проверив оба источника.**

## IKIGAI-опрос

Триггеры: "проведи ikigai опрос", "настрой цели", "ikigai"

Задать 8 вопросов строго по одному — ждать ответа после каждого:

1. "Расскажи о себе — чем занимаешься, что важно в жизни прямо сейчас?"
2. "Что любишь делать? От чего получаешь энергию?"
3. "В чём реально силён? Что получается лучше чем у других?"
4. "За что платят или готовы платить?"
5. "Что нужно людям от тебя? Какую проблему решаешь?"
6. [Сформулировать IKIGAI:] "Вот как я вижу твой IKIGAI: [формулировка]. Верно?"
7. "Где хочешь быть через 3 года? Конкретно — в работе, деньгах, жизни."
8. "Какие привычки хочешь отслеживать?"

После подтверждения создать/перезаписать:
- `goals/ikigai.md` — IKIGAI с 4 сферами
- `goals/vision-3y.md` — видение на 3 года
- `goals/habits.md` — трекер привычек

## Фильтр задач по IKIGAI

Триггеры: "что важнее всего", "с чего начать", "расставь приоритеты"

Прочитать `goals/ikigai.md` и `tasks/active.md`.
Классифицировать каждую задачу:
- К IKIGAI — ведёт к точке пересечения
- Полезная — нужна, но не к IKIGAI
- Пустая — не ведёт никуда

Вывести:
```
🎯 К IKIGAI (делать первыми):
1. Задача

🗒 Полезные:
1. Задача

⬇️ Можно отложить:
1. Задача
```
