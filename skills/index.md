# EVA Skills Index

**При каждом старте:** читай `memory/lessons.md`

Читай только нужный файл по задаче:

- `notion.md` — интеграция с Notion (DB ID, curl-шаблоны). Читай при работе с задачами.
- `jobs.md` — логика джобов (утро, проверки, вечер, рефлексия). Читай при выполнении джобов.
- `docs.md` — обработка документов из Telegram (PDF, фото, сканы). Читай когда приходит файл.
- `dialog.md` — триггеры диалога, поиск по wiki. Читай при обычных запросах.
- `ikigai.md` — интервью IKIGAI и Кайдзен. Читай при команде /ikigai или запросе про цели.
- `crm.md` — создание баз данных (клиенты, задачи, записи) через SQLite. Читай при любом запросе на создание базы или ведение учёта.
